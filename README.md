# Sprint 7 - SQL Data Collection

Data collection and analysis project using SQL queries to extract and explore data.

## What it does
- Writes SQL queries to extract data from databases
- Filters, groups and aggregates data using SQL
- Analyzes query results with Python and Pandas

## Skills demonstrated
- SQL queries (SELECT, WHERE, GROUP BY, JOIN)
- Data extraction and filtering
- Data analysis with Pandas
- Combining SQL and Python

## Tools
- Python 3
- Pandas
- SQL

## Context
Project developed during TripleTen Data Science Bootcamp - Sprint 7
